# capstone
Capstone Project for final term in Fanshawe


Technologies used


The full stack picture
Frontend (browser)
│
├── HTML
├── CSS
├── Bootstrap
├── JavaScript
└── React (optional)

Backend (server)
│
├── Python
├── Django
└── Django REST Framework

Database
│
└── PostgreSQL

Developer Tool
│
└── VS Code

HTML — Structure

Job:
Defines the structure of the page.

HTML is the skeleton.

HTML decides:
what exists
order of elements
forms
inputs

No styling
No logic

CSS — Styling

Job:
Controls how things look.

CSS decides:
colors
spacing
layout
fonts
responsive design
Bootstrap — Pre-built styling

Job:
Gives you ready-made CSS classes.

Bootstrap provides:
grids
buttons
forms
navbars
responsive layouts
JavaScript — Behavior

Job:
Makes the website interactive.

JS handles:
button clicks
form submission
fetch API calls
validation
logic

React — Frontend framework

Job:
Organizes JavaScript for large applications.

React helps with:
reusable components
state management
auto UI updates
complex frontend logic

React does NOT:
replace backend
store data
replace CSS

Python — Backend language

Job:
Programming language used on the server.

Python handles:
logic
calculations
conditions
data processing
Python alone cannot make a website.
It needs a framework.

Django — Backend framework

Job:
Turns Python into a web backend.

Django handles:
API endpoints
user authentication
permissions
business logic
security
database communication

Django REST Framework — API layer

Job:
Allows Django to send and receive JSON.
Frontend talks to backend using HTTP:
fetch("/api/users")

PostgreSQL — Database

Job:
Stores permanent data.

Stores:
users
passwords
posts
products
orders

Database keeps data even after server restarts.
Django talks to DB using ORM.

Frontend never talks to DB directly.

VS Code — Code editor

Job:

Used to:

write HTML
write CSS
write JS
write Python
manage Git
run terminal
